Отчет по лабораторной работе 1.1.4: Измерение интенсивности радиационного фона.¶

Цель работы:

В работе использовался: счетчик Гейгера-Мюллера, блок питания, компьютер.

Включали компьютер для совершения измерений для основного эксперимента. В результате демонстрационного эксперимента убеждаемся, что в результате увеличения числа измерений:

1) Измеряемая величина колеблется

2) Среднее значение постепенно выходит на определенную величину

3) Колебания величины погрешности отдельного измерения уменьшаются, погрешность одного измерения становится постоянной величиной

4) Колебания величины погрешности среднего значения уменьшаются, сама величина убывает

Переходим к основному эксперименту: измерение плотности потока космического излучения за 10 секунд.

Распределение Пуассона¶

Распределе́ние Пуассо́на — вероятностное распределение дискретного типа, определяемое следующей формулой: $$ P(k) = \frac{\lambda^k}{k!}e^{-\lambda}, $$ где $P(k)$ определяет веротность того, что произойдет $k$ событий, если в среднем проиходит $\lambda$ событий.

In [40]:
# Импортируем функцию вычисления факториала из стандартной математической библиотеки
from math import factorial


def poisson(k: int, mean: float):
    """
    Функция вычисляющая вероятность согласно распределению Пуассона
    k - число событий
    mean -среднее число событий
    """
    return (mean**k)*np.exp(-mean)/factorial(k)

# Векторизуем функцию- делаем её способной работать с числовыми массивами
poisson = np.vectorize(poisson)

Проверка принадлежности экспериментальной выборки к распределению Пуассона по критерию $\chi^2$ (критерию согласия Пирсона)¶

Критерий хи-квадрат — любая статистическая проверка гипотезы, в которой выборочное распределение критерия имеет распределение хи-квадрат при условии верности нулевой гипотезы.

Критерий согласия Пирсона, или критерий согласия $\chi^2$ — наиболее часто употребляемый критерий для проверки гипотезы о принадлежности наблюдаемой выборки $x_{1},x_{2},...,x_{n}$ объёмом $n$ некоторому теоретическому закону распределения $F(x,\theta )$. Свойства критерия были впервые исследованы Карлом Пирсоном в 1900 году.

Статистика критерия согласия $\chi ^{2}$ Пирсона определяется соотношением: $$ X^2_n = n \sum^k_{i=1}\frac{[n_i/n - P_i(\theta)]^2}{P_i(\theta)} $$ В случае проверки простой (однопараметрической) гипотезы в пределе при $ n\to \infty $ эта статистика подчиняется $ \chi _{r}^{2} $-распределению с $r=k-1$ степенями свободы, если верна проверяемая гипотеза.

# Импортируем гамма-функцию
from scipy.special import gamma

def chi2ProbabilityDensity(x, k=1):
    """
    Распределение хи-квадрат
    """
    return (0.5**(k/2)) * (x**( (k/2) - 1))*np.exp(-x/2)/gamma(k/2)

def chi2Poisson(data : np.ndarray, step=4):
    """
    Функция для расчета статистики критеря Пирсона
    Принимает:
    data - исходные данные
    step - шаг разбиения на интервалы
    Возвращает:
    Значение статистики
    Количесвто степеней свободы
    """
#     Гистаграмируем
    bins = np.arange(0, data.max()+step, step)
    hist = np.histogram(data, bins=bins)
    m = len(bins)
#     Вычисляем вероятности оказатся в интервале
    mean = data.mean() #
    poissonCumValue = np.zeros(m) #
    for i in range(m-1):
        for j in range(bins[i], bins[i+1]):
            poissonCumValue[i] += poisson(j, mean) #
    poissonCumValue[-1] = 1 - np.sum(poissonCumValue) #
#     print(poissonCumValue,np.sum(poissonCumValue))
    n = len(data) #
    ni = np.hstack((hist[0],[0])) #
    empiricProbability = ni/n #
    terms = ((empiricProbability - poissonCumValue) **2 )/poissonCumValue #

#     print(terms)
    return n*(np.sum(terms)), len(ni)-1
